Key DNS Concepts

Important ideas needed to understand the Domain Name System.

1. Domain Name System (DNS)

DNS is the system that translates human-friendly domain names into machine-readable IP addresses so users can reach websites and services on the internet.

2. IP Address

An IP address is the numerical network address used by computers to route traffic and identify a destination on the internet.

3. Recursive Resolver

A recursive resolver receives a DNS request from a client and performs the lookup process if the answer is not already cached.

4. Authoritative Name Server

An authoritative server stores the official DNS records for a domain and provides the final answer when that domain is queried.

5. Time to Live (TTL)

TTL is the value that determines how long a DNS response may be cached before it should be refreshed from an authoritative source.

6. DNSSEC

DNSSEC is a set of security extensions that helps verify DNS responses are authentic and have not been altered in transit.

7. DoH and DoT

DNS over HTTPS and DNS over TLS are privacy-focused protocols that encrypt DNS traffic between the client and resolver.

8. DNS Records

DNS records are entries in a DNS database that define how a domain name behaves, including where it points and what services it supports.

9. Root Servers

Root servers are the starting point of DNS resolution. They direct queries to the appropriate top-level domain servers.
