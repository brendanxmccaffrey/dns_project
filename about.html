About This Website

This website was created as part of Project Milestone #1 for IT 3203. It presents research from Assignment 1 in web page form using HTML and CSS. The topic of this project is the Domain Name System (DNS), a foundational technology that enables the internet by translating domain names into IP addresses.

The purpose of this project is to demonstrate fundamental web development skills, including page structure, navigation, and styling, while presenting technical information in a clear, organized, and accessible format.

Project Goals

  • Create a six-page informational website
  • Use HTML elements such as headings, lists, images, and tables
  • Use a single CSS file for layout and styling
  • Present research clearly for a general audience
